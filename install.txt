Scattertex

Instalando com o Anaconda:
conda install -c conda-forge scattertext

Instalando com o Pip:
pip install scattertext


Spacy

Instalando com o Anaconda:
conda install -c conda-forge spacy

Instalando com o Pip:
pip install -U spacy



Pandas

Instalando com o Anaconda:
conda install pandas

Instalando com o Pip:
pip install pandas
